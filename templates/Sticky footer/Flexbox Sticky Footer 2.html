Flexbox Sticky Footer

The simplest, hack-free way to make a sticky footer for modern web browsers.

  • 100% CSS
  • JavaScript Free

The Problem

On 'short' pages without much content, your footer rides up too high. I know. Embarassing, huh? This of course is not a new problem, but in the old-days before flexbox, we had to resort to using hacks or JavaScript to make our footers sticky. What a mess. Not any more.

The Fix

To use, make the body element the Flex Container and then add a flex-grow property to the main content area. No matter the content size, the div expands to take up any leftover space. See the commments in the CSS for more information.
